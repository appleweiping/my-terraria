# Third-Party Save Notices

The repository's `LICENSE` applies to original repository text, scripts, and original save work unless a file says otherwise.

Third-party Terraria worlds and players under `Terraria_saves/imported/` keep their original source-page license or permission status. They are not relicensed as MIT by this repository.

Each imported third-party save directory must include a `THIRD_PARTY_NOTICE.md` with:

- source URL
- source author
- retrieval date
- observed license or permission status
- source file id or download URL when available
- local file names
- SHA-256 hashes
- import decision

Current public imported saves:

| Directory | Source license observed |
|---|---|
| `Terraria_saves/imported/story-of-red-cloud/` | Public Domain |
| `Terraria_saves/imported/all-boss-battle-arenas/` | Creative Commons Attribution-NonCommercial 3.0 Unported |
| `Terraria_saves/imported/compact-all-items-hub-master/` | GNU General Public License version 3 (GPLv3) |

High-quality third-party saves whose source pages show restrictive or unclear redistribution terms are backed up locally under ignored `private-imports/` paths and are not committed to GitHub.
