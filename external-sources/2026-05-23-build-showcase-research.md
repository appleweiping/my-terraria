## Licensing Reality for Build Showcases

Build showcases are even harder to find with open licenses than adventure maps:
- Most builders share via Steam Workshop (non-redistributable)
- Forum posts default to All Rights Reserved
- CurseForge build maps rarely have explicit open licenses
- GitHub-hosted build worlds are rare

## Recommended Path Forward

1. **Manual CurseForge check** — Visit Khaiostomization page in browser to confirm license and check if author has updated it
2. **Contact KhaiosLive** — Request redistribution permission for Khaiostomization; it's the strongest candidate
3. **GitHub search pass** — Run targeted searches for pixel art / showcase worlds with MIT/CC licenses
4. **r/Terraria outreach** — Post asking if any builders have shared worlds with open redistribution rights
