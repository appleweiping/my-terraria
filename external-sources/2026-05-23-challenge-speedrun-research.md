# 2026-05-23 Challenge and Speedrun World Research

Research into Terraria challenge and speedrun worlds for Pillar 4 coverage. Currently have 1 entry (All Boss Battle Arenas Expert, CC BY-NC 3.0). Need 1 more.

## What Counts as Pillar 4

Per ROADMAP.md:
- Boss rush arenas (already covered by All Boss Battle Arenas Expert)
- Class-locked challenge setups
- Hardcore progression worlds
- Speedrun-optimized seeds with documented routes
- "No-hit" practice worlds
- Pre-built class loadouts for challenge runs

## Internal Candidates (Original Projects)

The Boss Rush Colosseum (now complete) is an original project in Pillar 4 territory. However, it is classified under Pillar 6 (Original Projects) rather than Pillar 4 (Challenge/Speedrun) since it was purpose-built for this archive.

**Decision**: Boss Rush Colosseum satisfies Pillar 4 as a secondary classification. Pillar 4 now has:
- All Boss Battle Arenas Expert (CC BY-NC 3.0, external import)
- Boss Rush Colosseum (original, Master Mode, 24 arenas)

**Pillar 4 target of 2+ is met** if we count the original project. If the standard requires 2 *external* imports, 1 more is needed.

## External Candidates

### 1. Seeded Challenge Worlds (GitHub)

Some speedrunners publish seeded world files with documented routes. These are rare but occasionally appear on GitHub.

Search targets:
- `terraria seed speedrun world github`
- `terraria challenge world wld github`
- `terraria getfixedboi seed world`

### 2. Class Challenge Worlds (CurseForge)

CurseForge has a "Challenges" category under Terraria maps. Key candidates to manually verify:
- Class-locked worlds (melee-only, summoner-only, etc.)
- Hardcore challenge setups
- "Calamity Death Mode" challenge worlds (modded)

### 3. Terraria Community Forums

The forums have dedicated challenge run threads. Some include downloadable world files. Most are All Rights Reserved but worth checking for explicit permission language.

## Recommended Path Forward

Given that Boss Rush Colosseum (original) covers Pillar 4 as a secondary classification, the immediate priority is lower than Pillars 3 and 5. However, an external challenge world would strengthen the pillar.

**Best candidate type**: A seeded world with a documented speedrun route, hosted on GitHub with an open license. These are rare but the most likely to have redistribution-compatible licensing.
