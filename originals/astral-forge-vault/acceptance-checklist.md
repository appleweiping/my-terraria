- [ ] `Astral_Forge_Vault.wld` exists and opens in Terraria.

- [ ] Optional matching `.map` cache exists for at least one curator/test character.

- [ ] Spawn hub is built and labeled.

- [ ] Class sanctums exist for melee, ranger, mage, summoner, and hybrid.

- [ ] Transit network connects spawn, oceans, dungeon, jungle, shimmer, underworld, sky, and major farm/arena zones.
- [ ] Signs or in-world labels make the map self-explanatory.
- [ ] The world still has personality and exploration value.

- [ ] Source/design notes are updated.
- [ ] Inventory includes the world and any paired players.

- [ ] Git LFS tracks all binary save files.
