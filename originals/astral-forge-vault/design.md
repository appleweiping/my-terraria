# Astral Forge Vault Design

## Core Identity

`Astral Forge Vault` should be a maximalist but playable Terraria archive world. It is not a pure cheat map and not a normal survival world. It should let a player do nearly anything useful from one world while still feeling like a coherent place.

## World Pillars

1. **Command Center**
   - Spawn is a compact operations hub, not a decorative lobby.
   - Must include crafting, storage, teleport routing, loadout access, arena access, and map labels.

2. **Vault Library**
   - Items are organized by use, not by chaotic chest spam.
   - Suggested wings: blocks, furniture, weapons, armor, accessories, potions, materials, boss/event summons, fishing, wiring/mechanics, vanity, banners/trophies.

3. **Class Sanctums**
   - Five ready-to-use class rooms: melee, ranger, mage, summoner, hybrid.
   - Each room should contain themed armor/weapon/accessory sets, buff stations, mannequins, and short notes in signs.

4. **Boss And Event Proving Grounds**
   - One universal boss arena.
   - Specialized arenas for Duke Fishron, Empress of Light, Moon Lord, Pumpkin/Frost Moon, Old One's Army, and invasions.
   - Arena support: campfires, heart lanterns, honey, bast statues, sunflower access, ammo/mana/potion staging, safe nurse access where appropriate.

5. **Farm Matrix**
   - Herb and potion farm.
   - Truffle worm farm.
   - Money/event farm.
   - Biome-key/material farms.
   - Fishing ponds for major biomes.

6. **Transit And Legibility**
   - Pylons, teleporters, minecart spine, hellevator, skybridge, ocean links.
   - Signage should make the world usable without a separate guide.

7. **Aesthetic Standard**
   - The hub should look like a celestial foundry: star-metal, glass, luminite-like palette, mechanical wiring, and biome exhibits.
   - Utility areas can be dense, but not ugly dumping grounds.

## Character Set Target

When original characters are created, they should be paired with the world:

| Character | Role | Requirements |
|---|---|---|
| Astral Warden | melee/tank | Endgame melee loadouts, mobility, survivability, arena testing kit. |
| Nebula Archivist | mage | Mana economy, burst and sustained magic setups, utility spells. |
| Vortex Curator | ranger | Bullet, arrow, rocket, and summon-ammo support. |
| Stardust Keeper | summoner | Minion/sentry setups, whip options, hybrid survivability. |
| Solar Courier | generalist | Building, wiring, mining, fishing, and transport kit. |

## Completion Bar

The finished save must be useful to a serious Terraria player. It should answer: "Can I open this world and immediately test builds, build new projects, fight bosses, farm materials, sort items, and understand where everything is?"

If the answer is not yes, the save is not complete.
